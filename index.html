Keyboard Shortcuts

  • Ctrl + Space - Play/Pause speech
  • Ctrl + S - Stop speech
  • Ctrl + + - Increase font size
  • Ctrl + - - Decrease font size
  • Tab - Navigate between controls
  • Esc - Close modal
